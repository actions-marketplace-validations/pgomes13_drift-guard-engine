# Installation

## Homebrew (macOS / Linux)

```sh
brew tap pgomes13/tap
brew install drift-guard
```

## Go install

```sh
go install github.com/pgomes13/drift-guard-engine/cmd/driftengine@latest
```

## Build from source

```sh
go build -o drift-guard ./cmd/driftengine
```

Or via Make:

```sh
make build
```
